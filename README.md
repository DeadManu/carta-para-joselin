# Archivos para reemplazar

Estos archivos ya tienen agregada la frase:

"Cada segundo que pasa me acerca al final de esta espera, pero nunca borrará lo que siento por ti. ❤️"

También incluyen la cuenta regresiva configurada con:

- 730 días
- 24 horas
- 60 segundos

## Cómo usarlos

Copia estos archivos dentro de tu proyecto y reemplaza los existentes:

- `index.html`
- `css/styles.css`
- `js/script.js`

Después sube los cambios a GitHub y Vercel actualizará la página.
