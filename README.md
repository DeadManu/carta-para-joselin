# Carta para Joselin

Este proyecto es una página web estática con una carta animada.

## Archivos principales

- `index.html`: estructura de la página.
- `css/styles.css`: diseño visual, colores, animaciones y versión para teléfono.
- `js/script.js`: animación del fondo, corazones flotantes y efectos.
- `assets/favicon.svg`: ícono de la página.
- `vercel.json`: configuración básica para Vercel.

## Cómo abrirla en tu computadora

1. Descomprime el archivo `.zip`.
2. Abre `index.html` con doble clic.
3. Se abrirá en el navegador.

## Cómo subirla a GitHub

1. Crea un repositorio nuevo en GitHub.
2. Sube todos los archivos y carpetas del proyecto, no solamente el ZIP.
3. Asegúrate de que `index.html` quede en la raíz del repositorio.

## Cómo publicarla en Vercel

1. Entra a Vercel.
2. Importa el repositorio de GitHub.
3. No necesitas seleccionar framework.
4. Déjalo como proyecto estático.
5. Publica el proyecto.
6. Vercel te dará un enlace para compartirlo en teléfono.

## Cómo editar el texto

Abre `index.html` y cambia el contenido de la carta dentro de:

```html
<div class="texto">
  ...
</div>
```

También puedes cambiar el nombre, fecha y firma dentro del mismo archivo.
